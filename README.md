## **설치 및 실행 방법**

### **사전 요구사항**
- Node.js (v18 이상)
- yarn

### **설치 과정**
```bash
# 프로젝트 클론
git clone https://github.com/Pop-py/poppy-fe.git

# 디렉토리 이동
cd poppy-fe

# 의존성 설치
yarn install

# 개발 서버 실행
yarn dev
```
